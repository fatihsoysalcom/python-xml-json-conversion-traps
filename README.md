# Python XML JSON Conversion Traps

This example demonstrates how to convert a complex XML string into a JSON object using Python's standard library. It highlights common 'traps' in XML-to-JSON conversion, such as preserving attributes, handling text content versus child elements, and correctly representing lists of repeating XML elements, to prevent data loss or inconsistencies.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
